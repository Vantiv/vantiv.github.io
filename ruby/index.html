Getting started

Initially, Matz looked at other languages to find an ideal syntax. Recalling his search, he said, “I wanted a scripting language that was more powerful than Perl, and more object-oriented than Python3.”

In Ruby, everything is an object. Every bit of information and code can be given their own properties and actions. Object-oriented programming calls properties by the name instance variables and actions are known as methods.

The Litle Ruby SDK follows the above principles providing a simple Ruby implementation of the Litle XML API. This SDK was created to make it as simple as possible to connect process your payments with Litle. This SDK utilizes the HTTPS protocol to securely connect to Litle, you can start right away by connecting to our Sandbox environment.
