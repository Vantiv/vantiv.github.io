PHP is a widely-used general-purpose scripting language that is especially suited for Web development. PHP is typically used in conjunction with the Apache Webserver.

Litle's PHP SDK makes it quick and easy to connect your PHP site to Litle's payment platform. Download our SDK, install it, and start testing against our sandbox environment right away.

Our source code is located on Github

Our package can be downloaded using Composer and is hosted at Packagist

Our PHP SDK supports all of the functionality present in Litle XML v8. Please see the online copy of our XSD for Litle XML to get more details on what is supported by the Litle payments engine.
